CDC Data Analysis

This project contains an R-based workflow for loading, cleaning, analyzing, and/or visualizing data from a large CSV file from the CDC
stored externally on Google Drive.

Because the dataset is too large to store directly in GitHub, the data file is hosted separately and can be downloaded here:

https://drive.google.com/file/d/1mjm3IFBdQ1o54FQ47tfUBBKi-KBjs7O4/view?usp=drive_link
